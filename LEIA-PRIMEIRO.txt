BIG — ARQUIVO DEFINITIVO DO APP

Projeto: Big
Criador: Colômbia
Empresa: Cl Inc. Enterteiments
Sistema alvo: antiX / Debian sem systemctl
Porta padrão: 8088
Execução: fork/simples, SEM cluster

ATENÇÃO:
Esta versão LOCAL contém credencial Google em:
credentials/google-service-account.json
Não suba este ZIP em repositório público.

INSTALAÇÃO DIRETA NO ANTIX:
1) unzip big-definitive-app-local-ready.zip
2) cd big-definitive-app
3) chmod +x *.sh scripts/*.js
4) sudo ./install.sh
5) big-menu

ACESSOS:
App principal: http://localhost:8088/member-app/
Painel admin : http://localhost:8088/admin-app/

GOOGLE SHEETS:

Para criar a planilha automaticamente:
cd /opt/big-server
node scripts/sheets-bootstrap.js

Ou pelo painel:
Painel Admin -> Google Sheets -> Inicializar planilha

TERMUX SEM ROOT:
Copie TERMUX-DEPLOY-NO-ROOT.sh para o Termux e rode:
bash TERMUX-DEPLOY-NO-ROOT.sh

SSH DO ANTIX CORE:
Para servir o arrumador de SSH pelo celular, rode no Termux:
bash TERMUX-SERVE-ANTIX-SSH-FIX.sh
Depois digite no antiX Core o comando curto exibido pelo Termux.
