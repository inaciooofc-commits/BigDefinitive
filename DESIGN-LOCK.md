# Big Design Lock

Esta versão atualiza o código para seguir a imagem de referência gerada para o projeto **Big**.

## O que foi travado no visual

- App principal com sidebar escura, logo cursiva neon, busca superior, carteira, notificações e perfil no topo.
- Home em grade premium com hero cyber beach, player lateral, sala em destaque, avisos, missões, ranking, jogos, loja e perfil.
- Painel admin com estética técnica, métricas horizontais, gráficos visuais, saúde do sistema, região, rotas privadas, configurações rápidas e atividade recente.
- Cores base: preto profundo, roxo neon, ciano, magenta e branco suave.
- Cards em glassmorphism com bordas neon sutis e sombras escuras.
- O app mantém conteúdo +16 elegante, sem nudez e sem erotização explícita.

## Arquivo de referência

A imagem usada como referência visual está em:

`docs/big-ui-reference.png`

O código não usa a imagem como fundo colado. A interface foi refeita em HTML/CSS para parecer com ela e continuar funcional.
